HTML Basics

Structure of an HTML Document

Understanding the anatomy of an HTML document is fundamental. The <!DOCTYPE html> declaration sets the HTML version, while the <html> element encompasses the entire document. The <head> section hosts metadata, and the <body> section contains the content. This structural foundation forms the backbone of well-formed HTML pages.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My HTML Document</title>
  </head>
  <body>
    <!-- Content goes here -- >
  </body>
</html>

HTML Tags and Elements

HTML relies on tags and elements to structure content. From simple paragraph tags <p> to headings <h1> through <h6>, links <a>, and image tags <img>, each element serves a specific purpose. Mastery of these foundational elements is crucial for creating compelling and well-structured web pages.

<p>This is a paragraph.</p>
<h1>This is a heading level 1</h1>
<a href="https://www.example.com">Visit Example</a>
<img src="image.jpg" alt="Description">

Text Formatting

Text formatting is an art within HTML. Emphasize text with <strong>, italicize with <em>, underline with <u>, and format code with <code>. These formatting tags add depth and clarity to your textual content.

<p><strong>Bold text</strong> and <em>italicized text</em>.</p>

Lists and Tables

Structuring information is pivotal for effective communication. HTML provides tools such as lists (<ul>, <ol>, <dl>) and tables (<table>, <tr>, <td>) to organize and present data. Mastering these elements contributes to creating clean, organized, and readable content.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<table>
    <tr>
        <td>Row 1, Column 1</td>
        <td>Row 1, Column 2</td>
    </tr>
</table>

Forms and Input Elements

Interactivity is the essence of many modern web applications, and HTML forms facilitate this. The <form> element, along with <input>, <select>, and <textarea>, empowers you to create dynamic and user-friendly experiences, collecting and processing user input seamlessly.

<form action="/submit" method="post">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username" required>
    <input type="submit" value="Submit">
</form>

Advanced HTML

Multimedia Elements

Elevate your web content with multimedia elements. The <audio> and <video> elements bring life to your pages. Attributes like controls provide users with playback control, while autoplay can set the tone for a dynamic user experience.

<audio controls>
    <source src="audio.mp3" type="audio/mp3">
    Your browser does not support the audio element.
</audio>

<video controls width="400">
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video element.
</video>

Semantic HTML

Understanding the semantics of HTML is crucial for accessibility and search engine optimization. Semantic tags like <article>, <section>, and <nav> provide meaning to your document structure, enhancing both user experience and search engine understanding.

<article>
    <h2>Article Title</h2>
    <p>Article content goes here.</p>
</article>

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
    </ul>
</nav>

HTML5 Features

HTML5 introduces cutting-edge features for a more engaging web experience. The <canvas> element opens the door to dynamic graphics, while <details> and <summary> provide a structured way to handle additional content. The <progress> element gives users insight into ongoing tasks.

<canvas id="myCanvas" width="200" height="100"></canvas>

<details>
    <summary>Show More</summary>
    <p>Hidden content revealed.</p>
</details>

<progress value="50" max="100">50%</progress>

Accessibility Best Practices

Create inclusive web experiences by implementing accessibility best practices. Leverage ARIA roles and attributes to ensure your web pages are accessible to users with diverse needs. Provide alternative text for images and ensure interactive elements are navigable.

<button aria-label="Close" onclick="closeDialog()">X</button>
<img src="profile.jpg" alt="Profile Picture" role="img" aria-label="John Doe's Profile Picture">
